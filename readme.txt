A helper plugin to contain your MainWP Dashboard customisation snippets.

== Description ==

The purpose of this pulugin is to contain your customization snippets for the [MainWP Dashboard](https://wordpress.org/plugins/mainwp/) plugin. This plugin will help you to keep all your changes in one location and independent of the other components. This will enable you to safly update your plugins and themes without of worry of losing your modifications as well as deactivating your customizations if needed for troubleshooting.

*Credit:* [Theme Customisations by WooThemes](https://github.com/woocommerce/theme-customisations) plugin which the MainWP Customisations plugin is inspired and built on.

== Installation ==

1. Upload `maiwnp-customisations` to the `/wp-content/plugins/` directory on your MainWP Dashboard site
2. Add your customisations to the plugin either locally or via Plugins > Editor in your MainWP Dashboard site
3. Activate the plugin through the 'Plugins' menu in WordPress

== Usage ==

* Add any CSS snippets to `/wp-content/plugins/maiwnp-customisations/custom/mainwp-custom-style.css`.
* Add any PHP snippets to `/wp-content/plugins/maiwnp-customisations/custom/functions.php`.
* Add any jQuery snippets to `/wp-content/plugins/maiwnp-customisations/custom/mainwp-custom-js.js`.
